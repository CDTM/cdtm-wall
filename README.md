* HTML 5

* Bootstrap
* jQuery
* Socket io


## Backend 

* Python
* Flask

## Deployment

Docker is used to build and deploy the app

use the `build_docker.sh` for building a container

use the `export_docker.sh` for exporting a .tar file

use the `load_docker.sh` for importing the .tar file
